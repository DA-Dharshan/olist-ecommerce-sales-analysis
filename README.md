# Olist E-Commerce Sales Analysis

## Project Overview

This project analyzes sales performance using the Olist Brazilian E-Commerce Dataset. The objective was to identify revenue trends, top-performing product categories, customer review patterns, regional performance, and shipping cost insights.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## Analysis Performed

* Revenue by Product Category
* Monthly Revenue Trend
* Revenue by State
* Customer Review Analysis
* Shipping Cost Analysis
* Average Review Score Analysis

## Dashboard Features

* Total Revenue KPI
* Total Orders KPI
* Average Review Score KPI
* Average Shipping Cost KPI
* Interactive Slicers
* Business Insights Section

## Key Findings

* Beauty Products generated the highest revenue.
* São Paulo was the highest-performing state.
* Most customers provided 5-star ratings.
* Average review score remained above 4.
* Revenue showed clear monthly trends.
